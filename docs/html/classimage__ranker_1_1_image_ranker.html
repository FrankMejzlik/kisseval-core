Detailed Description

Public API for evaluation library used for keyword-based models.

On instantiation it loads all cofigured data packs defined in the file at DATA_INFO_FPTH and preprocesses them for later usage. When instantiated it handles SEQUENTIAL requests on loaded models and imagesets. It also accepts submit request and stores them inside the database.
